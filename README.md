## Overview

This project demonstrates how YOLO can be used to detect and capture license plates from a stationary camera in real time. Detected plates can be logged and stored, providing a foundation for applications such as parking management, access control, traffic monitoring, and security systems.

In addition to license plate detection, this repository includes several computer vision examples, including:

* Pose Detection
* Hand Detection and Tracking
* Emotion Detection

These examples are intended to serve as practical learning resources and building blocks for developing more advanced YOLO-based computer vision applications. Whether you're exploring object detection for the first time or creating your own custom AI vision project, this repository provides a solid starting point.
